## 🚀 התקנה מהירה

### שלב 1: הורדה
1. הורד את כל הקבצים מהפרויקט הזה
2. שמור אותם בתיקייה על המחשב

### שלב 2: התקנה ב-Chrome
1. פתח Chrome ועבור ל: `chrome://extensions`
2. הפעל **Developer mode** (למעלה מימין)
3. לחץ **Load unpacked**
4. בחר את התיקייה שהורדת
5. ✅ התוסף מותקן!

### שלב 3: הגדרה (בחר אפשרות)

#### אפשרות 1: Chrome Local AI (מומלץ - חינמי לחלוטין!)
1. Chrome 127+ נדרש
2. הפעל: `chrome://flags/#prompt-api-for-gemini-nano`
3. הפעל מחדש את Chrome
4. **עובד offline, בלי API key!**

#### אפשרות 2: Google Gemini (חינמי!)
1. לך ל: [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. לחץ **"Get API Key"**
3. העתק את המפתח (מתחיל ב-`AIza...`)
4. הדבק בהגדרות התוסף → שמור
5. **1,500 בקשות ליום - בחינם!**

#### אפשרות 3: ChatGPT / Perplexity
- ChatGPT: [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
- Perplexity: [perplexity.ai/settings/api](https://www.perplexity.ai/settings/api)

## 🐛 בעיות נפוצות

### התוסף לא עובד?
1. ודא ש-Chrome עדכני (גרסה 127+)
2. בדוק שהכנסת API Key (אם לא משתמש ב-Chrome Local AI)
3. נסה לרענן את הדף

### Chrome Local AI לא עובד?
1. Chrome 127+ נדרש (Canary/Dev/Beta)
2. הפעל את הדגל: `chrome://flags/#prompt-api-for-gemini-nano`
3. הפעל מחדש את Chrome
4. השימוש הראשון - יוריד מודל (~1.7GB)

### הגעתי למגבלה היומית?
- החינמי מתחדש כל יום בחצות
- רואה טיימר בתוסף שמראה מתי מתחדש
- או שדרג ל-PRO לשימוש בלתי מוגבל
