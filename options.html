🤖 בחר ספק AI

בחר באיזה ספק להשתמש לתרגום וסיכום.

🖥️ Chrome Local AI (Gemini Nano)

AI מובנה בדפדפן Chrome — עובד ללא אינטרנט, ללא מפתח API, עם פרטיות מלאה. דורש Chrome 127+ עם הדגל chrome://flags/#prompt-api-for-gemini-nano.

בודק זמינות...
מגבלות: טקסט מוגבל ל-6,000 תווים. מודל קטן (Gemini Nano) – מתאים לשאלות קצרות וסיכומים.
🔧 הגדרות מתקדמות / מפתח API משלך (BYOK)

אופציונלי. המפתחות שלך — נשמרים רק במחשב שלך. אין אצלנו בשרת.

🔑 מפתח API – Google Gemini

ניתן לקבל מפתח API בחינם לגמרי (מכסה נדיבה מאוד!) ב- aistudio.google.com/apikey.
המודל: gemini-2.5-flash – מהיר, חכם ובחינם לשימוש רגיל.

✨ איך להוציא API Key בחינם:
  1. היכנס לחשבון Google שלך
  2. לך ל-Google AI Studio
  3. לחץ על "Get API Key" או "Create API Key"
  4. בחר פרויקט קיים או צור חדש (בחינם)
  5. העתק את המפתח (מתחיל ב-AIza...)
  6. הדבק למטה ולחץ שמור
💎 מה כלול בחינם: 15 בקשות/דקה • 1,500 בקשות/יום • 1M טוקנים/חודש • ללא כרטיס אשראי

🔑 מפתח API – OpenAI (ChatGPT)

ניתן לקבל מפתח API ב- platform.openai.com/api-keys.
המודל: gpt-4o-mini – מהיר, זול ואיכותי.

🔑 מפתח API – Perplexity

ניתן לקבל מפתח API ב- perplexity.ai/settings/api.

🎨 מראה

בחר מצב תצוגה מועדף.

📖 כיצד להשתמש

  1. בחר ספק AI ולחץ עליו (Chrome Local AI לא מצריך מפתח!).
  2. אם בחרת ספק ענן — הזן מפתח API ולחץ שמור.
  3. נווט לכל עמוד אינטרנט ולחץ על אייקון התוסף.
  4. בחר תרגם, סכם, או שאל.
  5. לייצוא היסטוריה לחץ 📥 במסך ההיסטוריה.
  6. קיצור מקלדת: Cmd+Shift+Y (Mac) / Ctrl+Shift+Y (Win).

💡 מידע

  • המפתחות נשמרים מקומית בלבד — אין שרת ביניים.
  • Chrome Local AI – חינמי, פרטי, ללא אינטרנט. מוגבל ל-6,000 תווים.
  • Gemini 2.5 Flash – מהיר מאוד, חינמי עם מכסה נדיבה.
  • ChatGPT (gpt-4o-mini) – איכות גבוהה, מחיר נמוך.
  • Perplexity Sonar – איכות גבוהה, דורש תשלום אחרי מכסה.
  • עמודים ארוכים נחתכים ל-48,000 תווים (6,000 ב-Local AI).
  • תוצאות נשמרות במטמון ל-24 שעות לכל עמוד.
